# LeanStock Backend

Production-grade inventory management backend built with:

- Express.js
- Prisma ORM
- PostgreSQL
- JWT Authentication
- RBAC Authorization
- Swagger/OpenAPI

## Features

- JWT auth
- Refresh tokens
- Role-based access control
- Email verification
- Password reset
- Product inventory management
- Pagination/filtering/sorting
- Swagger docs
